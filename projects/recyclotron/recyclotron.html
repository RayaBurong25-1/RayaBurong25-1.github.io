What

This project was thought of as part of the student entrepreneurship club LaunchX, to invent a product that would help solve the world's problems. It was sent to LaunchX's Global Pitch event, and advanced to the final round. It was presented at Massachusetts Institute of Technology as well. The project is an automatically sorting recycling bin, with camera and a classifier powered by machine learning, and motors to put the items into its corresponding compartment.

Where

This project was done in Phillips Exeter Academy, Exeter, NH.

When

December 2018 - March 2019 (4 months)

Why

With climate change becoming one of the world's pressing problems, recycling is becoming one of the more significant ways of life to help combat climate change as well. The problem is, recycling doesn't seem to be intuitive enough and there is not enough incentive for people to both with the "hassle". Thus, Recyclotron is born. There are two main parts of the contraption. First, the machine, which would automatically classify recyclable items, and would display a redeemable QR code. Second, the application, which could redeem the QR code to collect coins or use as a coupon with partner companies.

How

The machine was powered by a Raspberry Pi and its operation code is written in C++. The application, which I only had time to write a proof-of-concept, was coded in Java (Android).
